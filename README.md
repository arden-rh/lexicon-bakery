# Bakery

A modern bakery website built with React, TypeScript, and Vite. Browse cupcakes and wedding cakes with product details, ratings, and prices.

## Features

- Product catalog with cupcakes and wedding cakes
- Responsive design with SCSS modules
- React Router for navigation (Home, Products, About, Contact)
- TypeScript for type safety
- Product filtering and grid display
- Contact form with validation (demo - no backend)

## Tech Stack

- **React 19** with TypeScript
- **Vite** for build tooling
- **React Router** for routing
- **SCSS Modules** for styling
- **ESLint** for code quality

## Getting Started

### Install dependencies

```bash
npm install
```

### Run development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

### Preview production build

```bash
npm preview
```

## Project Structure

- `src/components/` - Reusable UI components
- `src/pages/` - Page components (Home, Products, About, Contact)
- `src/data/` - Product data
- `src/styles/` - Global styles and SCSS configuration
- `src/types/` - TypeScript type definitions
